# Design System Document

This document outlines the core design principles and token values for our digital products.

## Theme

- **Color Mode**: `light`
- **Headline Font**: `inter`
- **Body Font**: `inter`
- **Label Font**: `inter`
- **Roundedness**: `2` (Moderate)
- **Spacing**: `2` (Normal)

## Color Palette

Our brand leverages a distinct color palette to ensure consistency and brand recognition across all interfaces.

- **Primary Color**: `#1275e2` (Used for primary actions, main calls-to-action, and key interactive elements.)
- **Secondary Color**: `#5f78a3` (A supporting color for less prominent UI elements, chips, and secondary actions.)
- **Tertiary Color**: `#a957c6` (An additional accent color for highlights, badges, or decorative elements.)
- **Neutral Color**: `#74777f` (A foundational neutral base color for backgrounds, surfaces, and non-chromatic elements.)

## Typography

Our typography system is built to ensure readability and establish a clear visual hierarchy.

### Headlines
The `inter` font is used for all headlines, providing a modern and clear appearance.

### Body Text
The `inter` font is used for all body text, ensuring legibility and consistency across long-form content.

### Labels
The `inter` font is used for labels, ensuring consistency with the overall typographic system.

## Shape and Form

### Roundedness
Our UI elements feature a `moderate` level of roundedness (`2`), striking a balance between modern aesthetics and usability.

## Spacing

### Spacing Scale
We utilize a `normal` spacing scale (`2`) to define the whitespace within and between UI elements, promoting a balanced and readable layout.
