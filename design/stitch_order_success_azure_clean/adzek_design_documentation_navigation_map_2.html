# Adzek: Design Documentation & Navigation Map ## 1. Концепция бренда **Adzek** — это бренд бытовой химии, который относится к чистоте как к научной дисциплине. Мы называем это "Clinical Editorial". * **Слоган:** Chemistry refined for the home. * **Тональность:** Профессиональная, минималистичная, премиальная. --- ## 2. Карта страниц и их назначение ### 1. Homepage (Главная) * **Цель:** Погружение в бренд, демонстрация ключевых категорий (Laundry, Kitchen, Bathroom) и философии. * **Анимации:** Плавное появление (Fade-in) блоков при скролле. Параллакс-эффект для Hero-изображения. * **Пути:** Ведет в каталог, на страницу "О нас" и в раздел акций. ### 2. Product Catalog (Каталог) * **Цель:** Быстрый поиск и фильтрация продуктов. * **Анимации:** Мягкое увеличение карточки при наведении. Плавная фильтрация без перезагрузки страницы. * **Пути:** Клик по товару → Страница товара (PDP). Клик "Add to Cart" → Корзина. ### 3. Product Description Page (PDP - Карточка товара) * **Цель:** Детальное изучение продукта (состав, pH, аромат) и выбор объема. * **Анимации:** Смена изображений в галерее с эффектом cross-fade. Анимированные иконки преимуществ. * **Пути:** Кнопка "Add to Cart" → Переход в корзину. ### 4. Shopping Cart (Корзина) * **Цель:** Проверка списка покупок и расчет итоговой суммы. * **Анимации:** Плавное удаление товара из списка. Анимация изменения итоговой суммы. * **Пути:** Кнопка "Confirm & Pay" → Checkout. ### 5. Checkout (Оформление заказа) * **Цель:** Безопасный ввод данных для оплаты и доставки. * **Анимации:** Валидация полей ввода в реальном времени. Индикация процесса оплаты при нажатии на кнопку. * **Пути:** Кнопка "Confirm & Pay" → Order Success. ### 6. Order Success (Страница успеха) * **Цель:** Подтверждение оплаты и информация о доставке. * **Анимации:** Анимированная галочка "Success". * **Пути:** Кнопка "Back to Shop" → Homepage. ### 7. Personal Account (Личный кабинет) * **Цель:** Управление профилем, просмотр истории заказов и избранного. * **Анимации:** Плавное переключение между вкладками (Profile, Orders, Settings). * **Пути:** Ведет на страницы истории заказов и настроек. ### 8. Admin Dashboard (Панель администратора) * **Цель:** Внутренний инструмент для владельца бизнеса: аналитика продаж, управление стоком. * **Анимации:** Динамическое построение графиков продаж при загрузке. * **Пути:** Переход в раздел управления складом (Inventory Management). --- ## 3. Интерактивная логика Все страницы связаны через единый Header. Кнопки CTA (Call to Action) выделены фирменным цветом бренда для интуитивного понимания следующего шага в воронке продаж.
