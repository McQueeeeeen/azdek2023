# 08. Admin / Backoffice план

## MVP админка

- Каталог: CRUD товаров, вариантов, цен, медиа
- Заказы: список, фильтры, карточка заказа, статусы
- Платежи: статус, ручной retry link
- Доставка: метод, трек-номер, статус отправки

## Этап 2

- Промокоды и правила скидок
- Управление остатками
- B2B-заявки и approval flow
- Возвраты/рефанды
- Контент и юридические документы

## RBAC роли

- Owner
- Manager
- Support
- Content Editor
- Warehouse
