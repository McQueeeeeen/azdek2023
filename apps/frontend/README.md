# Frontend (Next.js)

Implemented storefront scaffold over backend contracts:

- `/catalog`
- `/catalog/[slug]`
- `/cart`
- `/checkout`
- `/order/[orderNumber]`
- `/login`
- `/account`

Environment:

- `NEXT_PUBLIC_API_URL=http://localhost:4000/v1`

Commands:

- `pnpm --filter @azdek/frontend dev`
- `pnpm --filter @azdek/frontend build`
